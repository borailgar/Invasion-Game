Thank you for downloading this package! :)
This package provide a pure procdeural method to generate infinite, non-repetition starry sky texture.
Well-suited if you need render a real starry sky.

The "StarrySky" script component attached on Main Camera GameObject provide 2 variables:
 => Velociy: control the movement of the starry sky.
 => Enable: means enable/disable the post process starry sky render.

Also there are many shader variables exposed on material inspector panel.
Try tweak them and get the required visual appearance you like.
